## 🧠 Sobre o Projeto

O **Sistema de Produtividade da LK Soluções de T.I** é uma plataforma completa voltada para **organização pessoal, gestão de tarefas e planejamento estratégico**.

A ideia do projeto é reunir em um único ambiente diversas ferramentas utilizadas por profissionais, estudantes e empreendedores para melhorar **foco, planejamento e execução de atividades**.

O sistema será construído com uma arquitetura moderna utilizando **Node.js, Express e PostgreSQL**, além de um frontend leve e performático em **HTML, CSS e JavaScript**.

---

## 🎯 Objetivo

Criar uma **central de produtividade completa**, onde o usuário consiga:

* Organizar tarefas
* Planejar metas
* Estruturar projetos
* Melhorar o foco
* Acompanhar produtividade

Tudo em uma única plataforma.

# ⚙️ Como Executar o Projeto

## 1️⃣ Clonar o repositório

```
git clone https://github.com/seu-repositorio/sistema-produtividade-lk.git
```

---

## 2️⃣ Entrar na pasta do projeto

```
cd sistema-produtividade-lk
```

---

## 3️⃣ Instalar dependências

```
npm install
```

---

## 4️⃣ Configurar variáveis de ambiente

Crie o arquivo `.env`

```
PORT=3000

DB_HOST=localhost
DB_PORT=5432
DB_USER=postgres
DB_PASSWORD=sua_senha
DB_NAME=lk_produtividade
```

---

## 5️⃣ Rodar o servidor

```
npm start
```
